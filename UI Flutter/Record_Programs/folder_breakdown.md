Folder Breakdown of Flutter Project (Theory)

Main Folders in a Flutter Project:

android/ – Code for Android platform.
ios/ – Code for iOS platform.
lib/ – Main Dart source files (e.g., main.dart).
test/ – Unit testing files.
build/ – Auto-generated build files.
web/ – Web app-related files.
linux/ – For Linux desktop app support.
windows/ – For Windows desktop app support.
macos/ – For macOS desktop app support.
.idea/ – Android Studio settings.
.dart_tool/ – Dart environment tools and configs.
pubspec.yaml – Project dependencies file.
pubspec.lock – Locked dependency versions.
